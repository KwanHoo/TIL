# _PostgreSQL 소문자_

## 대소문자 구분

- PostgreSQL은 소문자가 기본임
- PostgreSQL에서는 기본적으로 모든 쿼리가 소문자로 변환됨
  - 다른 DB는 대문자가 기본일수가 있음
  - 컬럼이나 테이블명에서는 대소문자 구별이 없어서 쿼리 짤 때 대문자로 쓰든 소문자로 쓰든 상관이 없었으나 PostgreSQL은 소문자가 기본임으로 주의 해야함
